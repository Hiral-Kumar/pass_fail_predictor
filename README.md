# Student Pass/Fail Prediction System

A Machine Learning based web application that predicts whether a student is likely to **Pass or Fail** based on academic indicators.

The system not only predicts the result but also provides **human-readable explanations and improvement suggestions**.

---

## Features

- Predicts Pass/Fail using **Logistic Regression**
- Detects **risk factors** affecting academic performance
- Provides **reason of failure**
- Suggests **scope of improvement**
- Highlights **attendance shortage risk**
- Interactive web application built with **Streamlit**

---

## Machine Learning Model

The prediction system uses:

**Logistic Regression Classifier**

Why Logistic Regression?

- Simple and interpretable
- Suitable for binary classification
- Easy to deploy in lightweight applications

---

## Input Features

The model considers the following parameters:

- Attendance Percentage
- Internal Marks
- Assignment Score
- Study Hours per Week
